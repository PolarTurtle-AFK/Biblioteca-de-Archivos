# Biblioteca de Archivos v2

Página web simple para visualizar y descargar imágenes desde una biblioteca organizada en tarjetas.

## Características

* Visualización de imágenes
* Información básica del archivo
* Botón de descarga
* Enlace al inicio y correo de contacto

## Tecnologías

* HTML5
* CSS3

## Uso

1. Descargar el proyecto.
2. Abrir `index.html` en un navegador.
3. Usar el botón "Descargar" para guardar las imágenes.

## Estructura

```
/Proyecto
│── index.html
│── style.css
│── /img
     ├── asombrao.jpg
     ├── Alejandro.webp
     ├── Bocchi.jpg
```
